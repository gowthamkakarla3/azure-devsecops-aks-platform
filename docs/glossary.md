# 📘 Glossary (DevOps Terms Explained)

---

## AKS

Azure Kubernetes Service – Managed Kubernetes cluster

---

## ACR

Azure Container Registry – Stores Docker images

---

## Docker

Tool used to create containers

---

## Kubernetes

Container orchestration platform

---

## Pod

Smallest unit in Kubernetes (runs containers)

---

## Deployment

Manages pods in Kubernetes

---

## Service

Exposes application inside Kubernetes

---

## Ingress

Routes external traffic to services

---

## CI/CD

Continuous Integration / Continuous Deployment

---

## Key Vault

Secure storage for secrets

---

## LoadBalancer

Exposes service to internet

---

## Namespace

Logical separation in Kubernetes

---
