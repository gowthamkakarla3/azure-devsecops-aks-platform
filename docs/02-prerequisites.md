# 📌 Prerequisites (Before Starting)

## 💻 System Requirements

* Windows / Mac / Linux
* Minimum 8GB RAM recommended

---

## 🛠️ Install Required Tools

### 1. Azure CLI

Download:
https://learn.microsoft.com/en-us/cli/azure/install-azure-cli

Verify:

```
az version
```

---

### 2. Terraform

Download:
https://developer.hashicorp.com/terraform/downloads

Verify:

```
terraform -version
```

---

### 3. Docker Desktop

Download:
https://www.docker.com/products/docker-desktop/

Verify:

```
docker --version
```

---

### 4. kubectl

Install:

```
az aks install-cli
```

Verify:

```
kubectl version --client
```

---

## 🔐 Azure Account

* Create Azure account
* Ensure you have **subscription access**

---

## 🧠 Folder Setup

Clone repo:

```
git clone <your-repo>
cd azure-devsecops-aks-platform
```
