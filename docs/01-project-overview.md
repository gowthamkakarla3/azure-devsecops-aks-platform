# 🚀 Project Overview – Azure DevSecOps AKS Platform

---

## 📌 What is this project?

This project demonstrates how to build a **production-grade DevSecOps platform on Azure** using modern tools like:

* Kubernetes (AKS)
* Azure DevOps (CI/CD)
* Docker (Containerization)
* Azure Key Vault (Secrets)
* Azure Monitor (Observability)

---

## 🎯 Objective

The goal of this project is to simulate how real-world companies:

* Deploy scalable applications
* Automate deployments
* Secure secrets
* Monitor systems
* Use microservices architecture

---

## 🧠 High-Level Flow

```text
User → LoadBalancer → Ingress → Frontend → API Gateway → Microservices → AKS → ACR → CI/CD → Monitoring
```

---

## 🧱 Architecture Components

### ☁️ Infrastructure

* Azure Resource Group
* Virtual Network (VNet)
* Public & Private Subnets

---

### ☸️ Kubernetes (AKS)

* Private AKS Cluster
* Deployments & Services
* Horizontal Pod Autoscaler (HPA)

---

### 🐳 Containerization

* Docker images for each service
* Stored in Azure Container Registry (ACR)

---

### 🌐 Networking

* Azure LoadBalancer
* NGINX Ingress Controller
* Path-based routing

### 🔁 CI/CD

* Azure DevOps Pipeline
* Automated build & deployment

---

### 🔐 Security

* Azure Key Vault
* Managed Identity
* RBAC

---

### 📊 Monitoring

* Azure Monitor
* Log Analytics
